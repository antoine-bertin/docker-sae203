Commandes principales pour utiliser git (en ligne de commandes) ?

git status

Vérifie l'état du git

git log

Cette commande permet d'affichier l'historique des modifications

git add

Ajoute tous les fichiers modifiés

git commit -m "commentaire"

Elle sert non seulement a soumettre le ficher, ainsi que d'y mettre un commentaire pour le log

git push

Cela sert à envoyer les fichiers

git pull

Cela sert à mettre à jour son répertoire avec le git

git clone

Cette commande copie tout le repository distant sur le pc

git fetch

Récupére le contenu sans fusioner les branches

git reset

Cela permet de supprimer le dernier commit

git revert

Cela permet d'échanger 2 commit entre eux

git branch

Liste toutes les branches locales

git checkout NomDeBranche

Sert à changer de branche

git checkout -b NomDeBranche

Crée et passe sur nouvelle branche

git bisect

Cette commande permet de diviser les commits en deux afin de savoir qu'elle moitié fonctionne

git cherry-pick

cette commande permet de passer un numéro de commit et de le déplacer sur une autre branche
